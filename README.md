# AIProject-Automated-Video-to-Audio-Processing-Pipeline
Overview
The Automated Video to Audio Processing Pipeline is a Python-based project designed to convert YouTube videos into audio files effortlessly. This pipeline extracts audio from videos, transcribes the audio to text using AssemblyAI, and then converts the transcribed text back into audio using Google Text-to-Speech (gTTS). It serves as a valuable tool for content creators and researchers who wish to easily access audio formats of video content.

Features
Audio Extraction: Download audio from YouTube videos.
Transcription: Convert audio to text using AssemblyAI.
Text-to-Audio Conversion: Generate audio files from transcribed text using gTTS.
User-Friendly: Easy to use with minimal setup.
Requirements
Python 3.x
Libraries:
assemblyai
gtts
pydub
requests
IPython
